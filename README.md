RTL for Bootstrap-Sass
======================

**bootstrap-sass-rtl** is a another Bootstrap 3.x rtl version that born based on
[Bootstrap Sass](<http://github.com/twbs/bootstrap-sass>), with helping sass
variables in the peace and calm.
